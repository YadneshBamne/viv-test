# VIV SDK — Introduction

Welcome to **VIV**, your gateway to seamless integration with powerful Large Language Models (LLMs). VIV is built to empower developers with simple, flexible, and production-ready tools for embedding conversational AI into applications, products, and workflows.

Whether you're building a chatbot, a support assistant, or an intelligent automation tool, **VIV's SDK** helps you do it faster and smarter.

---

## What is VIV?

**VIV** is an AI-first platform focused on delivering customizable LLM-based solutions for a wide range of industries. With our robust API and developer-friendly SDK, you can integrate the power of LLMs directly into your app—on web, mobile, or server-side.

---

## ✨ Features

- 🔧 **Plug & Play SDK** — Start with minimal setup.  
- 📡 **Real-time Interactions** — Connect and stream responses from our LLMs.  
- 🔐 **Secure Auth** — Built-in support for token-based authentication.  
- 🧩 **Modular Design** — Choose your tools: chat, embeddings, generation, and more.  
- 📁 **Lightweight & Fast** — Designed for performance in production.  

---

## 📦 What You Get

- **VIV SDK**: A lightweight JavaScript/TypeScript package to call and stream LLM responses.  
- **API Access**: Secure endpoints for prompt completion, chat, and custom models.  
- **Customization**: Set system prompts, memory strategies, and temperature.  
- **Tooling**: Optional plugins for vector stores, logging, and analytics.  

---

## 🔑 Requirements

- A valid **VIV API Key**  
- **Node.js v16+** or a modern browser  
- **Internet connectivity** to access VIV LLMs
